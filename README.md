# Python-blog-image-tool

## 概要 Description
はてなブログ向け画像ツール

## 特徴 Features
はてなブログの各記事に掲載する写真や画像を変換して、アップロードして、画像のURLをコピーできる。


【特徴】
- 画像のサイズ変更ができる
- 画像のミラー反転ができる
- 画像を左または右に回転できる
- 画像に文字透かしを入れられる
- 画像を はてなフォトライフにアップロードできる
- アップロードした画像のURLをコピーできる
- 設定を保存できる

## 依存関係 Requirement

- Python 3.8.5  
- image_resize_sig	1.0.1
- fotolifeUpload	1.0.1
